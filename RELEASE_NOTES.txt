# LabelKeeper Release Notes

## v3.3.2 (2025-02-05)
- Device sync: Share data between devices using 6-character code
- PWA support: Install as app on iOS/Android/Desktop
- System dark mode: Respects device preference on first visit
- Bug fixes: Deep link navigation, QR code error messages

## v3.3.1 (2025-02-05)
- Fixed deep link navigation to sheets
- Added system dark mode detection

## v3.3.0 (2025-02-04)
- Name badges: Avery 5395 support (8 per sheet)
- Tent cards: Avery 5302 support with fold line (4 per sheet)
- Sheet labels: Print identification label on back with QR code
- Content linking: Option to link wizard content to sheets for reprinting
- OCR fix: Modal now appears above editor
- Paste button: Added to Full Editor toolbar

## v3.2.0 (2025-02-03)
- Print centering fix
- Split remaining labels feature
- Clipboard import with Excel support

## v3.1.0 (2025-02-02)
- Emoji picker
- OCR scanning with Tesseract.js
- Inventory breakdown display
- History improvements
